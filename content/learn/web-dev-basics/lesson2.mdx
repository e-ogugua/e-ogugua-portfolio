# Building Your First Web Page

Now that you understand the basics of how the web works, let's dive deeper into creating your first proper web page with HTML and CSS.

## HTML Document Structure

Every HTML document follows a standard structure:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Web Page</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <!-- Your content goes here -->
</body>
</html>
```

### Key Elements:
- `<!DOCTYPE html>`: Declares this is an HTML5 document
- `<html>`: The root element
- `<head>`: Contains meta information and links to resources
- `<body>`: Contains the visible page content

## Essential HTML Elements

### Text Elements
```html
<h1>Main Heading</h1>
<h2>Subheading</h2>
<p>This is a paragraph of text. It can contain <strong>bold</strong> and <em>italic</em> text.</p>
```

### Links and Images
```html
<a href="https://example.com">Visit Example</a>
<img src="image.jpg" alt="Description of image">
```

### Lists
```html
<ul>
    <li>Unordered list item</li>
    <li>Another item</li>
</ul>

<ol>
    <li>First item</li>
    <li>Second item</li>
</ol>
```

## Adding Style with CSS

Create a new file called `styles.css`:

```css
/* Reset default margins and padding */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

/* Style the body */
body {
    font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
    line-height: 1.6;
    color: #333;
    max-width: 800px;
    margin: 0 auto;
    padding: 20px;
}

/* Style headings */
h1 {
    color: #2c3e50;
    margin-bottom: 20px;
}

/* Style links */
a {
    color: #3498db;
    text-decoration: none;
}

a:hover {
    text-decoration: underline;
}

/* Add some spacing */
p {
    margin-bottom: 15px;
}
```

## Building a Simple Layout

Let's create a basic blog post layout:

```html
<article>
    <header>
        <h1>My First Blog Post</h1>
        <p class="date">Published on March 10, 2025</p>
    </header>
    
    <section>
        <p>This is the introduction to my first blog post. I'm learning web development!</p>
        
        <h2>What I've Learned So Far</h2>
        <ul>
            <li>HTML structure</li>
            <li>Basic CSS styling</li>
            <li>How to link files</li>
        </ul>
    </section>
    
    <footer>
        <p>Written by [Your Name]</p>
    </footer>
</article>
```

## Making It Responsive

Add these media queries to your CSS to make the layout responsive:

```css
/* Responsive design */
@media (max-width: 600px) {
    body {
        padding: 10px;
        font-size: 16px;
    }
    
    h1 {
        font-size: 24px;
    }
}
```

## Testing Your Page

1. Save your HTML file as `index.html`
2. Save your CSS in the same folder as `styles.css`
3. Open `index.html` in your web browser
4. Try resizing your browser window to see the responsive design in action

## Best Practices

1. **Semantic HTML**: Use appropriate tags (`<header>`, `<nav>`, `<main>`, `<footer>`, etc.)
2. **Accessibility**: Always include `alt` text for images and use proper heading hierarchy
3. **Performance**: Keep your CSS and JavaScript in separate files
4. **Validation**: Use the [W3C Markup Validation Service](https://validator.w3.org/) to check your HTML
